# 什么是决策树（Decision Tree）？

决策树是一种树形结构的模型，每一个节点表示对一个特征的判断，每一个分支表示判断结果，每一个叶子节点则对应一个预测输出（类别或数值）。

### 决策树的基本结构
<pre>
          [是否下雨？]
          /        \
      是              否
     /                 \
[是否有伞？]         [出门]
  /      \
有        没有
/           \
[出门]     [不出门]
</pre>

#### How this becomes a tree Decision Tree 是怎么“长成一棵树”的?

The split you just looked at is one node. To build a full tree, recurse: for each branch that's still mixed, run the same picking procedure on the rows that landed there. Stop when you hit a base case.

01 majority_vote
Most common label in a subset. The prediction at a leaf. “当前subset里最多的label” 
比如：当前subset label
买
买
买
不买

最多的是：买
那么如果最终停在这里leaf prediction = 买。


02 feature_accuracy / mutual_information
Score how good a split on a feature would be.
“评估哪个feature最适合切”比如：
年龄
收入
地区

计算：在 decision tree 里，当前节点要这样选 split, 选 mutual information 最大的那个 feature
x3?
/ \
x3=0 x3=1

然后你把数据分成两堆：

左边：x3 = 0 的 rows

只看这些 rows 里面的 y。

如果全是同一个 y，比如全是 0：

x3=0 → predict y=0

这个分支就停止，变成 leaf。

右边：x3 = 1 的 rows

如果 y 还混着 0 和 1, 继续在这一小堆数据里，重新计算剩下 features 的 mutual information, 然后选更大的那个继续 split。

所以训练树的逻辑是：

1. 在当前数据 subset 上计算每个 feature 和 y 的 mutual information
2. 选 MI 最大的 feature
3. 用这个 feature split 数据
4. 对每个 branch：
   - 如果 y 已经 pure，停止，给 leaf label
   - 如果还 mixed，继续重复 1-3

关键点：

MI 不是只在 root 算一次

每 split 一次以后，到了子节点，要在那个 subset 里重新算 MI。


03 optimize_split
Pick the feature with the highest score.
“真正选择最佳feature”比如年龄得分最高。那么当前node年龄。


04 train
Recurse on each branch until base case fires.

The four base cases Recursion has to stop somewhere. train returns a leaf when:

（1）The subset is empty. 当前subset没数据了。比如某个branch：没人落进去。

（2）All rows in the subset share the same label (pure — nothing to split). 100%纯 entropy=0。

（3）Every remaining feature is constant in this subset (no split possible). feature已经没法继续分了。例如feature全一样，但label不同。你已经没有任何切法了只能majority vote。

<pre>
年龄	收入	label
年轻	高	买
年轻	高	不买
</pre>

（4）The depth limit (max_depth) is reached — regularization to avoid overfitting. 这个特别重要：regularization（正则化）什么意思？如果树无限分，每个人都单独一个leaf，训练集100%准确。但泛化极差这叫overfitting（过拟合）

所以：人为限制max_depth = 5树不能太深。
