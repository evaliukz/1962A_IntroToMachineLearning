# 1. 为什么会有 Perceptron

假设有二维数据：

| -- | -- | ----- |

| 4  | 7  | 1     |

最简单想法，画一条线：w1*​x1​+w2​*x2​+b=0 例如 x1​+x2​−5=0，这条线把平面分成两边：这就是Decision Boundary

# 5. 神经网络历史上最经典的一个问题：为什么 XOR 需要 Multi-Layer Perceptron (MLP)？

### XOR（异或）：

两个输入不同 -> 输出1

两个输入相同 -> 输出0

为什么单层 Perceptron 不行？ Perceptron 就是一条线：w1​x1​+w2​x2​+b=0 只能一刀切。

○     ●

●     ○

但是 XOR：无论怎么画一条直线都切不开。 所以： XOR is NOT linearly separable.

<img width="505" height="245" alt="image" src="https://github.com/user-attachments/assets/3689eee6-3e5b-4eb8-bbe4-d9b69f07be00" />

### Hidden Layer 在干什么

图里面：输入：

x1

x2

隐藏层：

h1

h2

输出：

y

Hidden Layer 的作用：不是直接分类。而是先创造新的特征。

图中的 H(z) 右上角：σ(z)=H(z)​ 这不是 Sigmoid。这是：Heaviside Step Function， 就是最原始的 Perceptron。Sigmoid有e。

### h1 在算什么

图中：h1​=H(x1​+x2​+b1​)， b1​=−0.5，所以 h1​=H(x1​+x2​−0.5)

输入：(0,0)， 0+0−0.5=−0.5  输出：0

输入：(1,0)， 1+0−0.5=0.5  输出：1

输入：(0,1) 输出：1

输入：(1,1) 输出：1

所以：h1学会了：至少有一个1 即： OR

### h2 在算什么

图中：h2 = H(x1​+x2​−1.5)

测试：(0,0) -> 0

(1,0) -> 0

(0,1) -> 0

(1,1) -> 1

所以：h2学会了：两个都是1 即：AND

### Output Layer

最后：y=H(h1​−2h2​+b3​)

其中：b3 =−0.5

即：y=H(h1​−2h2​−0.5)

输入：(0,0) OR=0 AND=0 0−0−0.5<0  输出：0

输入：(1,0) OR=1 AND=0 1−0−0.5=0.5 输出：1

输入：(0,1) 输出：1

输入：(1,1) OR=1 AND=1 1−2−0.5=−1.5 输出：0

成功实现 XOR！

### 为什么 MLP 可以解决 XOR

其实：Hidden Layer先学习：OR, AND

Output： 再组合： OR - AND  即：XOR = OR AND NOT(AND)

神经网络不是魔法。

它是在做：

输入
↓

第一层：
学习简单规则

↓

第二层：
组合简单规则

↓

得到复杂规则

这实际上就是深度学习最核心的思想：

多层网络 = 不断创造新的特征，然后组合这些特征来解决复杂问题。

	​

#### 什么叫自己学？

假设做猫狗分类。

训练集：

图片1  猫

图片2  狗

图片3  猫

...

最开始：网络里面的参数都是*随机*的。

w2=-0.55

w3=2.1

...

这些数字完全乱的。

网络预测：

第一张猫：预测：Dog 90%

答案：Cat，错了。

于是计算 Loss：

例如：Cross Entropy：Loss=5.3

然后：Backpropagation：告诉每个 weight：你应该大一点，你应该小一点，你影响了错误

例如： w1 = 0.123 -> 0.135,  w2 -0.55 -> -0.48

然后继续训练几百万次。

最后：这些 weight 变成：非常适合识别猫狗

这就叫：网络自己学出来

实际上：Gradient Descent + 不断修改weight

#### 为什么 Activation 很重要？

假设没有 activation：

第一层：h=W1x

代入：y=W2W1x

Activation：

例如：

Sigmoid：Linear -> Nonlinear -> Linear -> Nonlinear

这样才能：学习复杂特征。

最经典的 XOR

没人教它：先算OR 再算AND

Gradient Descent 自动找到这一组权重。
