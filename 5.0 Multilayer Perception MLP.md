# 1. 为什么会有 Perceptron

假设有二维数据：
| x1 | x2 | Label |
| -- | -- | ----- |
| 1  | 1  | 0     |
| 5  | 6  | 1     |
| 4  | 7  | 1     |

希望预测：y∈{0,1}

最简单想法，画一条线：w1*​x1​+w2​*x2​+b=0 例如 x1​+x2​−5=0，这条线把平面分成两边：这就是Decision Boundary

# 2. Perceptron是什么

Perceptron：

<img width="290" height="99" alt="image" src="https://github.com/user-attachments/assets/e69e4b59-88db-4881-85bb-61a06d6765ba" />

即先算：z=w.T x+b

然后：y=step(z) 

step函数：z>0 → 1 , z≤0 → 0

所以：Perceptron = Linear Combination + Threshold

3. Perceptron的问题

Step Function：

突然跳跃

例如：z=−0.00001

输出：0

z=0.00001

输出：1

突然变了。

求导：d​step(z)/dz 不存在， 而 Gradient Descent 需要导数。所以：Step Function不能训练
