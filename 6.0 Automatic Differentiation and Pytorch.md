# What is Automatic Differentiation?

Automatic Differentiation 就是：电脑把计算过程记下来，然后按照链式法则（Chain Rule）倒着一步一步把梯度传回来。

Automatic Differentiation computes exact derivatives by decomposing a function into elementary operations and repeatedly applying the chain rule 
on the computational graph. Deep learning frameworks such as PyTorch use reverse-mode automatic differentiation (backpropagation) to efficiently 
compute gradients of the loss with respect to all model parameters.

# 一、先说它到底是干什么的

假设老板说：这个月利润下降了，你去查查到底是谁的锅。一个公司：

原材料价格 -> 生产成本 -> 产品售价 -> 销售利润 -> 最后利润下降了。

老板问：如果原材料涨价 1 块钱，利润会下降多少？这就是求导。

### 方法一：暴力试（数值求导）

你可以试：

原材料100元
利润500元

原材料101元
利润495元

利润下降5元

所以：

导数≈-5

这叫 Numerical Differentiation。

优点：简单

缺点：不精确, 要反复试

### 方法二：数学老师的方法（符号求导）

把整个利润公式写出来：Profit=(Price−Cost)×Sales

然后一步一步推公式。

优点：精确

缺点：公式越来越长。

### 方法三：Automatic Differentiation

它说：我根本不推整个大公式。

我只记录：

第一步干了什么

第二步干了什么

第三步干了什么

最后再倒着算回来。


#### 举一个超级简单例子

假设：y=(x+3)^2

比如 x=2

先正常计算。

第一步：a=x+3

得到：a=5

第二步：y=a²

得到：y=25

计算图：

x=2 -> +3 -> a=5 -> y=a²平方 -> y=25 **这叫 Forward Pass（正向计算）**。

现在要求导, 老板问：如果 x 增加一点点，y 会增加多少？

也就是：dy/dx

Automatic Differentiation 不重新推公式。它倒着走。

最后一步：y=a²

老师问：如果 a 增加1，y 增加多少？

我们知道：dy/da=2a

因为a=5 所以：dy/da=10

记录：a变化1, y变化10

再往前：a=x+3 如果 x 增加1，a 增加多少？

当然：da/dx=1

连起来：x变化1  a变化1 y变化10

所以：dy/dx=10×1=10, 这就是 Chain Rule。

#### 为什么叫 Automatic（自动）?

因为电脑会自动记录：

第一步

第二步

第三步

然后自动倒回来。

#### 神经网络其实一样

假设你的网络：

x -> W0x+b0 -> sigmoid -> z -> W1z+b1 -> sigmoid -> ŷ -> Loss

Forward：

第一步：W0x+b0

第二步：sigmoid

第三步：W1z+b1

第四步：sigmoid

第五步：Loss

电脑把每一步都记下来。

然后开始倒着走：Loss -> sigmoid -> Linear -> sigmoid -> Linear 一路乘回来。

#### 为什么叫 Computational Graph（计算图）？

把整个神经网络看成水管：

水源
 ↓
阀门A
 ↓
阀门B
 ↓
阀门C
 ↓
水龙头

现在发现：

水龙头出水太小。

老板问：

到底哪个阀门影响最大？

Automatic Differentiation：

先从水龙头开始。

问：

最后一个阀门影响多少？

再问：

倒数第二个阀门影响多少？

一直追溯到最开始。

这就是 Backpropagation。
