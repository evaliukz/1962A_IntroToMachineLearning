# CNN 的开山之作： Backpropagation Applied to Handwritten Zip Code Recognition (1989)

作者：Yann LeCun 等（后来的图灵奖获得者）

> ⭐⭐⭐⭐⭐ CNN 开山之作（LeNet 前身）
>
> **一句话总结：**
>
> 这篇论文第一次证明了：**通过局部连接（Local Receptive Field）+ 权重共享（Weight Sharing）+ Feature Map，可以直接从原始图片学习特征，而不需要人工设计特征。**

---

# 一、为什么会有 CNN？

## 当时的问题

1989 年，大多数图像识别系统：

```text
Image
   │
人工 Feature Engineering
   │
MLP / Classifier
   │
Prediction
```

工程师需要自己设计：

* 边缘
* 圆
* 横线
* 宽高比
* 纹理

神经网络只是负责分类。

问题：

> **Feature 全靠人工设计。**

---

## LeCun 的想法

为什么不能：

```text
Image
   │
Neural Network
   │
Prediction
```

让网络：

**自己学习 Feature。**

这就是 CNN 诞生的原因。

# 五、CNN 为什么比 MLP 好？

## MLP

收到图片：

```text
111000010101...
```

它不知道：

哪些像素：

* 相邻
* 属于眼睛
* 属于耳朵

位置一变：

数字全变。

因此：

泛化很差。

---

## CNN

LeCun 发现：

图片最大的特点：

> **局部区域比整体更重要。**

例如：

猫：

先看：

```text
眼睛

耳朵

鼻子
```

再组合：

```text
猫
```

因此：

CNN：

不是一次看整张图片。

而是：

```text
一个小窗口

↓

滑动

↓

整张扫描
```

# 六、CNN 四个核心创新

---

## 1. Local Receptive Field（局部感受野）

普通神经网络：

```text
每个神经元

↓

连接整张图片
```

CNN：

一个神经元：

只看：

```text
5×5
```

区域。

例如：

```text
□□□□□

■■■■■

□□□□□
```

像：

放大镜。

好处：

* 参数少
* 学局部特征

---

## 2. Weight Sharing（权重共享）

论文最大的创新。

假设：

Kernel：

学会：

```text
横线。
```

那么：

左边：

有横线。

右边：

有横线。

为什么：

重新学？

不用。

因此：

同一套：

Kernel

整张图片：

重复使用。

所以：

参数：

少很多。

例如：

论文：

64000+

Connections

但是：

只有：

9760

Independent Parameters。

原因：

Weight Sharing。

---

## 3. Feature Map（特征图）

一个 Kernel：

对应：

一种 Feature。

例如：

Kernel：

```text
检测横线
```

输出：

```text
哪里有横线。
```

Kernel：

```text
检测圆
```

输出：

```text
哪里有圆。
```

所以：

```text
12 Kernels

↓

12 Feature Maps
```

Feature Map：

本质：

就是：

Kernel 的检测结果。

---

## 4. Hierarchical Feature Learning（层次特征学习）

第一层：

学习：

```text
边缘

线条

角
```

第二层：

学习：

```text
眼睛

耳朵

鼻子
```

第三层：

学习：

```text
猫脸
```

最后：

```text
猫
```

即：

```text
低级特征

↓

高级特征

↓

目标识别
```

论文：

# 十、论文真正贡献

很多人说：

CNN 的贡献：

是：

卷积。

其实：

真正贡献：

有四个：

① Local Receptive Field

只看局部。

---

② Weight Sharing

一套参数：

整张图片共享。

---

③ Feature Maps

不同 Kernel：

学习不同 Feature。

---

④ Hierarchical Learning

边缘

↓

纹理

↓

部件

↓

目标

| --------------------- | --------------------------- |
| Convolution           | ✅ 保留                        |
| Weight Sharing        | ✅ 保留                        |
| Feature Maps          | ✅ 保留                        |
| Local Receptive Field | ✅ 保留                        |

# 十二、我学到什么（自己的理解）

这篇论文真正想告诉我们的，不是"如何做卷积"，而是：

> **图片识别不应该让网络直接学习整个图片，而应该利用图片的空间结构，从局部开始，逐层学习越来越复杂的特征。**

CNN 并不是直接学习"猫"。

而是：

```text
边缘

↓

线条

↓

圆

↓

眼睛

↓

耳朵

↓

猫脸

↓

猫
```

这也是今天几乎所有视觉模型（LeNet、AlexNet、VGG、ResNet，甚至 Vision Transformer 的 Patch Embedding）仍然遵循的核心思想。

---

# 一句话总结

> **Backpropagation Applied to Handwritten Zip Code Recognition (1989) 首次将局部感受野（Local Receptive Field）、权重共享（Weight Sharing）和特征图（Feature Map）结合到神经网络中，使模型能够直接从原始图像自动学习局部特征，并逐层组合成高级语义特征，奠定了现代卷积神经网络（CNN）的基础。**
