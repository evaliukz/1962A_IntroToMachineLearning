# CNN - Convolution（卷积）

卷积就是用一个小的可学习滤波器（Kernel）在整张图片上滑动，对每个局部区域做“乘积求和”，从而提取局部特征（如边缘、纹理、角点），再逐层组合成越来越高级的语义特征，最终识别整个物体。

## 为什么需要卷积？

普通神经网络（MLP）会把图片展开成一维向量：

```text
1 1 1 0 0
0 1 1 1 0
...

↓

111001110...
```

这样会丢失图片最重要的信息：

> **像素之间的空间关系（Spatial Information）**

例如：

```text
******
******
```

和

```text
* * *
* * *
```

拥有相同数量的像素，但表示完全不同的图案。


# Convolution（卷积）是什么？

卷积就是：

> **使用一个小的 Filter（Kernel）在图片上滑动，对局部区域进行计算，从而提取图片特征。**

Filter（Kernel）：

```text
1 0 1
0 1 0
1 0 1
```

它可以理解为一个小窗口（Sliding Window）。

---

# 卷积计算过程

假设图片局部区域：

```text
1 1 1
0 1 1
0 0 1
```

Kernel：

```text
1 0 1
0 1 0
1 0 1
```

步骤：

1. 对应元素相乘（Element-wise Multiplication）

```text
1×1
1×0
1×1

0×0
1×1
1×0

0×1
0×0
1×1
```

2. 全部相加（Summation）

```text
1+1+1+1 = 4
```

得到输出：

```text
4
```

Kernel 再向右滑动一格，重复上述过程。

最终得到 Feature Map：

```text
4 3 4
2 4 3
2 3 4
```


# 什么是 Feature Map？

卷积后的输出称为：

- Feature Map
- Convolved Feature

它表示：

> 图片中每个局部区域与当前 Filter 的匹配程度。

数值越大，说明当前位置越符合该 Filter 所寻找的特征。



# 为什么叫 Sliding Window？

Kernel 会不断在图片上滑动（Slide）：

```text
□□□□□
■■■

↓

 □■■■

↓

  □■■■
```

每移动一步，就计算一次卷积。

因此称为 **Sliding Window**。



# CNN 学到了什么？

CNN 不会直接识别"猫"。

而是逐层学习越来越复杂的特征。

第一层：

- Edge（边缘）
- Line（直线）

第二层：

- Corner（角）
- Texture（纹理）

第三层：

- Eye（眼睛）
- Nose（鼻子）
- Ear（耳朵）

最后：

- Cat（猫）

即：

> **低级特征 → 高级特征 → 整体目标**



# Filter 是谁设计的？

PPT 中：

```text
1 0 1
0 1 0
1 0 1
```

只是示例。

实际训练中：

Filter 的参数全部由神经网络学习得到，例如：

```text
0.32  -0.84   1.15
-0.53  0.71   0.28
1.02  -0.16   0.45
```

不同 Filter 会自动学习检测不同特征，例如：

- Horizontal Edge（水平边缘）
- Vertical Edge（垂直边缘）
- Diagonal Edge（斜边）
- Texture（纹理）
- Circle（圆形）

通常一个 CNN 会学习几十甚至几百个 Filter。

例如：

```text
64 Filters
```

则会输出：

```text
64 Feature Maps
```



# CNN 为什么参数少？

普通全连接网络：

假设图片：

```text
100 × 100
```

输入：

```text
10000 个像素
```

连接到：

```text
1000 个神经元
```

参数：

```text
10000 × 1000

≈ 1000 万参数
```

CNN：

一个 3×3 Kernel：

```text
3 × 3 = 9 个参数
```

这 9 个参数会在整张图片上重复使用（Weight Sharing）。

因此：

- 参数数量极少
- 更容易训练
- 更不容易过拟合



# CNN 的优势

✅ 保留图片空间信息（Spatial Information）

✅ 提取局部特征（Local Features）

✅ 参数共享（Weight Sharing）

✅ 参数量远小于全连接网络

✅ 能逐层学习复杂语义特征
