# SoftMax

<img width="1536" height="1024" alt="image" src="https://github.com/user-attachments/assets/9db5dceb-717b-415b-bcc1-c7cded0af72e" />


当y出来的不是一个value，而是多个value，比如：

Cat
Dog
Rabbit
Horse

网络最后输出：

z=[2.1,1.3,0.2,-0.5]

这些叫：logits它们不是概率。

为什么要softmax来 normalize？ 

Softmax normalize the output so that it can be interpreted probabilistically

意思：希望变成：

Cat      0.60

Dog      0.27

Rabbit   0.09

Horse    0.04

满足：

都大于0
都小于1
加起来等于1


# L2 normalization

### 什么是 L2 normalization?

假设网络输出：

y=(3,4).T

先计算：L2 norm：∥y∥2​=sqrt(3^2+4^2)=5

然后除以它：y^​ =(3/5,4/5).T

满足(3/5)^2+(4/5)^2=1

Softmax: 除以所有exp和 -> 概率和等于1

L2 normalization: 除以向量长度 -> 向量长度等于1

二者共同特点：

都不是独立输出

都会让gradient互相影响
