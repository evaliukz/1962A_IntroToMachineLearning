# SoftMax

<img width="1536" height="1024" alt="image" src="https://github.com/user-attachments/assets/9db5dceb-717b-415b-bcc1-c7cded0af72e" />


当y出来的不是一个value，而是多个value，比如：

Cat
Dog
Rabbit
Horse

网络最后输出：

z=[2.1,1.3,0.2,-0.5]

这些叫：logits它们不是概率。

为什么要softmax来 normalize？ 

Softmax normalize the output so that it can be interpreted probabilistically

意思：希望变成：

Cat      0.60

Dog      0.27

Rabbit   0.09

Horse    0.04

满足：

都大于0
都小于1
加起来等于1

这样才能理解成：

60%是猫

27%是狗
