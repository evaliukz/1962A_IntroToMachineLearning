| 写法                     | 意思                          |
| ---------------------- | --------------------------- |
| `requires_grad=True`   | 我要对这个变量求导                   |
| `torch.xxx`            | 必须用 PyTorch 函数，不能用 `np.xxx` |
| `@`                    | 矩阵乘法                        |
| `.backward()`          | 自动反向传播                      |
| `.grad`                | 梯度存在这里                      |
| `.item()`              | 把单个 tensor 变成普通数字           |
| `with torch.no_grad()` | 更新参数时不要记录计算图                |
| `W0.grad[0,0]`         | 数学里的 (W_{11})               |
