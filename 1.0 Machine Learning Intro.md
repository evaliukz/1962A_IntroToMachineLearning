# What is Machine Learning?

A computer program is said to learn from
experience E with respect to some class of
tasks T, and performance measure P, if its
performance at tasks in T, as measured by P,
improves with experience E. - Tom Mitchell
(Machine Learning 1997)

● Example: (easy)
○ T := Predict the next word someone will type.
○ P := Is the word correct or not?
○ E:= A list of texts made by the= perhaps


● Example: (hard)
○ T := Autonomously drive a vehicle
○ P := Follows the rules of the road, avoids obstacles, reach destination
○ E := Senor data from human and AI  driven cars.

# The Supervised Machine Learning Setting
● Given a dataset D
● We want to approximate (more on this later) the target function  y = c(x)
● By a function in our hypothesis space y = h(x)

# Different types of target functions
● Binary Classification
● Multiclass Classification
● Ordinal Classification
● Regression
● Ranking
● Structured Output
● Multilabe

| Task Type                     | 输出是什么    | 例子                            | 常见 Loss                              |
| ----------------------------- | -------- | ----------------------------- | ------------------------------------ |
| **Binary Classification**     | 2个类别中的一个 | 垃圾邮件/非垃圾邮件；猫/狗                | Binary Cross Entropy (BCE)           |
| **Multiclass Classification** | 多个类别中的一个 | 手写数字0-9；图片分类1000类             | Cross Entropy                        |
| **Ordinal Classification**    | 有顺序的离散类别 | 电影评分1⭐-5⭐；风险等级Low/Medium/High | Ordinal Loss / Cross Entropy         |
| **Regression**                | 连续数值     | 房价预测；温度预测                     | MSE、MAE                              |
| **Ranking**                   | 排序结果     | Google搜索结果；推荐系统排序             | Pairwise Ranking Loss、LambdaRank     |
| **Structured Output**         | 复杂结构     | 机器翻译；语音转文字；代码生成               | Sequence Cross Entropy               |
| **Multilabel Classification** | 同时属于多个类别 | 一张图片同时有「狗」「草地」「人」             | Binary Cross Entropy (每个标签一个sigmoid) |
